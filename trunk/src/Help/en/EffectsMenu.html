Effects

This menu contains many commands that can be used for applying special effects to your image.

Plugins

Paint.NET supports adding additional effects by way of plugins, which you can download and install. More information can be found at the plugins section of the forum.

Examples

The following base image will be used to show what each effect (except for Red Eye Removal) is capable of:

  • Repeat [effect name]

    If you have run an effect before, you may use this command to run it again with the same settings.

  • Blurs submenu

    The Blurs submenu provides several effects for various types of blurring:

  • Gaussian Blur

  • Median

  • Motion Blur

  • Radial Blur

  • Unfocus

  • Zoom Blur
